# InvestEdge
Full Stack Trading Dashboard.A comprehensive, full-stack stock trading web application inspired by Zerodha Kite. This project simulates real-time equity trading, portfolio tracking, and order management with a highly interactive and responsive React frontend and a secure Node.js backend.

* Implemented secure user authentication and authorization with Passport.js and JWT.
* Built a dynamic watchlist with stock search, filtering, and quick buy/sell actions.
* Created a custom trade execution system with automatic portfolio updates and weighted average cost calculation.
* Added inventory validation to prevent users from selling more shares than they own.
* Designed an interactive dashboard displaying margin, investments, holdings, and real-time P&L tracking.
* Utilized React Context API for instant UI updates after trade execution without manual page refreshes.
* Integrated Chart.js to visualize portfolio allocation and performance through interactive charts.
* Developed a responsive and user-friendly interface with support for Dark Mode and Light Mode themes.
* Designed RESTful APIs and MongoDB schemas for managing users, orders, holdings, and positions.
